# Git & Version Control

This folder contains notes about Git version control system and related concepts.

## Files in this folder:

- **`branch.md`** - Git branch merging guide with commands for switching branches, pulling updates, merging, and pushing changes
- **`flags.md`** - Git command flags reference including `-M`, `-m`, and `-u` (set upstream) explanations
- **`git-merging.md`** - Comprehensive guide to Git merging including conflict resolution, fast-forward merges, and handling uncommitted changes
- **`rebasing.md`** - Notes about rebasing vs merging (work in progress)

## Topics Covered:

- Branch management and merging workflows
- Conflict resolution strategies
- Git command flags and options
- Upstream branch configuration
- Fast-forward vs regular merges
- Stashing and handling uncommitted changes

Perfect for developers who want to understand Git workflows beyond basic push/pull operations.
