# Learning Resources & Development Skills

This folder contains notes about learning strategies, useful resources, and professional development skills.

## Files in this folder:

- **`advanced-topics.md`** - Roadmap of advanced topics to explore including OS, networking, concurrency, software design, databases, and monitoring tools
- **`cool-links.md`** - Collection of useful links and resources
- **`links.md`** - Curated list of helpful GitHub repositories and learning resources
- **`system-design-vs-ai.md`** - Comparison between system design and AI approaches
- **`underrated-skills.md`** - Essential but underrated programming skills including shell commands, Git mastery, build systems, and problem-solving approaches
- **`using-ai-correctly.md`** - Best practices for using AI in development including feature definition, planning, and the 3T principle (Tiny, Testable, Total)

## Topics Covered:

- Advanced computer science topics and learning roadmaps
- Essential programming skills beyond coding
- Shell command mastery and system administration
- Git advanced features (rebase, stash, reflog)
- Build systems and development workflows
- AI-assisted development best practices
- Problem-solving methodologies
- Curated learning resources and tools
