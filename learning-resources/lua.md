# lua

- its a lightweight, and fast language
- its pretty simple so its not that that hard to learn
- its very fast so it can be embedded into the game
- has 21 keywords (making it easy to learn)
- luarocks is the package manager
- it is a dynamic language (so no type annotations)
- supports functional programming patterns (so you can feed functions into functions)
- language is single threaded
- with coroutines, you can pause and resume functions
- you can run c in lua, and lua in c
